## Getting Started

1. Ensure Jupyter is installed (included in requirements.txt)
2. Launch Jupyter:
   ```bash
   jupyter notebook
   ```
3. Navigate to the notebooks directory
4. Open desired notebook

## Best Practices

- Save outputs for reproducibility
- Document assumptions and decisions
- Keep notebooks focused on single topics
- Clean up before committing (clear outputs if sensitive)
