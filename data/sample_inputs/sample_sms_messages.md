# Sample M-Pesa SMS Messages

This directory contains sample SMS messages for testing the SMS parser tool.

## Money Received

```
RB12KLM Confirmed. You have received Ksh5,000.00 from JOHN DOE 254712345678 on 18/11/2025 at 10:30 AM. New M-PESA balance is Ksh15,000.00. Transaction cost, Ksh0.00.
```
